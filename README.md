# CAGT-Net-DDoS-Detection
Cross-Attention Guided Transformer Network for DDoS Attack Detection
## Overview

This repository contains the implementation of the proposed CAGT-Net framework for binary Distributed Denial-of-Service (DDoS) attack detection using the CIC-DDoS2019 dataset.

## Dataset

- Dataset: CIC-DDoS2019
- Number of Features: 80
- Classification: Binary (Benign vs DDoS)

## Requirements

- Python 3.10+
- TensorFlow
- NumPy
- Pandas
- Scikit-learn
- Matplotlib

## Files

- DDoS_Attack.ipynb : Complete implementation
- Dataset.zip : Dataset used for training and testing

## Performance

Accuracy : 98.56%
Precision : 98.58%
Recall : 98.56%
F1-score : 98.55%
